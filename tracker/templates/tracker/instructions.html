User Instructions

Introduction

Welcome to the Telegram Message Tracker system. This application allows you to monitor and manage messages from various Telegram channels, process them, and forward them to designated targets.

About This Guide

This comprehensive guide will walk you through all the features of the Telegram Message Tracker system. Whether you're a new user or looking to explore advanced features, you'll find detailed instructions and helpful tips throughout this documentation.

Key Features

Listeners

Set up listeners to monitor specific Telegram channels and collect messages automatically.

Targets

Configure target channels where processed messages will be forwarded.

Posts

Review, edit, approve, or reject messages before they are forwarded to targets.

Automation

Create rules to automatically process messages based on specific criteria.

Managing Listeners

Listeners are connections to Telegram channels that you want to monitor. Here's how to set up and manage listeners:

Creating a New Listener

1
Access Listeners Page

Navigate to the "Listeners" section from the main navigation menu.

2
Add New Source channel

Click the "Add Listener" button to open the creation form.

3
Enter Channel Details

Fill in the required information:

  • Channel Name: A descriptive name for your reference
  • Channel URL: The public URL of the Telegram channel (e.g., https://t.me/channelname)
  • Active following: Change to active if you want to start listening
4
Save and Verify

Click "Save" to create the listener. The system will verify the channel exists and is accessible.

Service will automatically start listening to the channel and collecting messages.

Managing Existing Listeners

You can edit, pause, or delete listeners as needed:

Editing

Click the "Edit" button next to a listener to modify its settings.

Pausing

Toggle the "Active following" switch to temporarily pause monitoring.

Deleting

Click the "Delete" button to permanently remove a listener.

Working with Targets

Targets are the Telegram channels where processed messages will be forwarded. Here's how to set up and manage targets:

  1. Navigate to the "Targets" section from the main menu
  2. Click the "Add Target channel" button
  3. Enter a name for the target
  4. Provide the Telegram channel URL where messages will be forwarded
  5. Set permissions for the bot in the target channel (must have posting rights)
  6. Click "Save" to create the target

Your Telegram bot must be added as an administrator to the target channel with posting permissions.

To establish a connection between a listener and a target:

  1. Go to the "Listeners" page
  2. Select a listener from the list
  3. Click on "Linked Targets" tab
  4. Click "Link Target" button
  5. Select one or more targets from the dropdown menu
  6. Click "Save" to establish the connection

Processing Posts

The Posts section is where you review and manage messages collected by your listeners before they are forwarded to targets.

Post Management Interface

Post Actions
  • Post

    Approve and forward the message to linked targets

  • Reject

    Discard the message (will not be forwarded)

  • Edit

    Modify the message content before posting

Post Statuses
  • Pending

    New messages awaiting review

  • Posted

    Messages that have been approved and forwarded

  • Rejected

    Messages that have been discarded

Filtering and Searching Posts

Use the search and filter options to find specific posts:

  • Search by content: Enter keywords in the search box
  • Filter by status: Use the dropdown to show only Pending, Posted, or Rejected messages
  • Filter by date: Use the date range selector to find messages from a specific period
  • Filter by source: Select a specific listener to see only its messages
